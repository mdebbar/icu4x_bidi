# icu4x_bidi

This is an optimized WASM build of the Bidi bits of ICU4X.

## Building

#### WASM

To build the WASM module:

```
$ npm run build-wasm
```

#### JavaScript wrappers

To bundle the JavaScript wrappers:

```
$ npm run build-js
```

#### Everything

To build everything at once:

```
$ npm run build
```
